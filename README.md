# Student-Attendence-Tracker

A Java-based console application designed to manage and track student attendance efficiently.
The System allows admin to add students, mark daily attendance, generate attendance reports and identify defaulters on attendance percentage

#Features
- Add and manage student details
- Mark daily attendance (Present/Absent)
- Prevent duplicate attendance entries
- Generate daily and monthly attendance reports
- Identify defaulters (attendance < 75%)
- Admin-only access for modifying record

#Technologies
  - Java
  - OOP Concepts
  - File Handling
